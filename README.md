## Лендинг для методист онлайн-курсов

### Описание проекта
Лендинг разработан для методиста онлайн-курсов

### Ключевые особенности
- Чистый и современный дизайн, ориентированный на пользовательский опыт
- Адаптивная верстка для всех устройств (мобильные, планшеты, десктоп)
- Быстрая загрузка страниц благодаря оптимизации Next.js
- Удобная навигация и структура контента

### Стек технологий
- **Next.js**
- **SCSS**
- **TypeScript**
